#  Python Password Strength Checker

A simple Python-based tool that checks the strength of a password based on common security rules.  
This project helps users create stronger passwords and demonstrates basic cybersecurity concepts using Python.

---

## Features
- Checks minimum password length  
- Validates presence of:
  - Uppercase letters  
  - Lowercase letters  
  - Numbers  
  - Special characters  
- Detects spaces in passwords  
- Provides clear feedback on password strength  

---

##  Technologies Used
- Python 3
- Regular Expressions (`re` module)
